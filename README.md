# UNP_PROJECT_BI_WITH_POWER_BI_BANK_DATA_ANALYSIS


In this project, I collaborated with UNP to build an interactive banking dashboard using Microsoft Power BI. I structured the dashboard across multiple sections covering overall financial performance, customer demographics, transaction trends, and branch-level analysis.
Through the analysis, I observed patterns in customer distribution, transaction behavior over time, and differences in branch performance. The data indicated trends such as a limited customer base spread, uneven contribution across branches, and fluctuations in transaction activity.
I used these insights to identify key business challenges and provided data-driven recommendations, focusing on how the bank could improve customer engagement, balance branch performance, and strengthen transaction consistency. The project emphasizes not just visualization, but the ability to interpret data and translate it into actionable business decisions.
